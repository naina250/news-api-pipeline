# News APT Data Pipeline

Built my first ETL pipeline using Python, in which I performed
- Extracted live news data from NewsAPI
- Transformed the JSON response
- Loaded it into a CSV file

## Tech Used
- Python
- requests
- csv
- NewsAPI

## How to Run
1. Get free API key from https://newsapi.org
2. Install requests: pip install requests
3. Run: python api_2.py

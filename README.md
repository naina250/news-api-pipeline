# API Data Pipeline

## Project 1 — News API Data Pipeline
Built my first ETL pipeline using Python, in which I performed
- Extracted live news data from NewsAPI
- Transformed the JSON response
- Loaded it into a CSV file

## Tech Used
- Python
- requests
- csv
- NewsAPI

---

## Project 2 — YouTube Data Pipeline
Built a second ETL pipeline using RapidAPI, in which I performed
- Extracted YouTube channel videos using POST request
- Transformed nested JSON response
- Loaded it into a CSV file

### Tech Used
- Python
- requests
- csv
- RapidAPI (YouTube138)

---

## How to Run
1. Get API keys from https://newsapi.org and https://rapidapi.com
2. pip install requests
3. Run api_2.py for news data
4. Run api_3.py for YouTube data
